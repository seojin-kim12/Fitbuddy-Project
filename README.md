## **운동 기록 서비스**

**건강과 운동에 관심이 높아지는 현대사회에 "<span style="color:yellow;">FITBUDDY</span>"서비스는 개인의 운동 활동을 체계적으로 기록하고 목표를 설정하는 데 효과적인 서비스**

**사용자들은 다양한 운동 정보를 손쉽게 공유할 수 있고 서로간의 경쟁과 협력을 통한 동기부여를 주는 서비스**

주요 기능
1. 운동 커뮤니티
2. 운동 일정 기록
3. 프로필
4. 로그인 및 회원가입
